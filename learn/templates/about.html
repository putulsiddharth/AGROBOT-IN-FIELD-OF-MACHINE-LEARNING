About this Project

This is what I do.

The proposed project could be divided into 3 main parts. At it's centre the project is a crop recommendation system using various characteristics like geographical location, soil chemistry, weather condition etc. Then comes the tips and basic learning material for user who want to grow the crop recommended by the system.

Agro Infotainment content like blogs will be made available as an introduction to agriculture and farming of the 21st century. Finally it is intended to club all of these components into a website to provide a seamless UI experience to the user.

Tools used: Scikit-Learn,Jupyter Notebook, Visual studio code.Techniques used: Classification, Machine Learning, HTML, CSS, JavaScript, Jquery, JSON, Flask.
